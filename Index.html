Visual Studio Code is a great platform to create websites! To download and use it, here's what you need to do: First you need to download Visual studio code using this link: https://code.visualstudio.com/Download. Then you need to choose which operating system (OS) you use. Once the installer is installed, click on it. Next you need to agree to their policy and then keep clicking next. Lastly, click finish and... voila! Visual Studio Code is now on your device! For a more in-depth tutorial, watch this video: https://www.youtube.com/watch?v=nfqMzlaX_d4

The console screen is a screen for the programmer's reference. To print a variable on the console screen, you need to use the command console.log() and put the variable in the parentheses (()) Example: console.log(variable_example). If you want to print something that isn't a variable, you need to use the command console.log() and put what you want to put that isn't a variable within quotations ("") in parentheses (()) Example: console.log("example") Kindly watch this short video before moving forward: https://www.youtube.com/watch?v=pKsvDf-sJQE. When you are on the browser page viewing your preview, right click anywhere on the page and click inspect or just use the keyboard shortcut which is ctrl + shift + I. Then click on "console" which is near the top next to "elements".
